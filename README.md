# 📊 Super Sales Store Analytics - Power BI Project

This repository contains an interactive **Power BI dashboard** developed for analyzing Super Sales Store data. The project includes two labs focused on sales insights and customer analytics.

## 🚀 Key Highlights

- Built an interactive dashboard using key KPIs:
  - Total Sales  
  - Profit  
  - Quantity Sold  
  - Profit Margin  

- Performed data cleaning and transformation for accurate analysis  

- Used **DAX (Data Analysis Expressions)** to create calculated measures and enhance insights  

- Created interactive visualizations:
  - Charts, Maps, Filters, and Slicers  
  - Regional sales trends  
  - Product performance  
  - Customer purchasing behavior  

- Applied forecasting techniques to identify future sales trends and support data-driven decision making  

## 📂 Files Included

- 📄 **PDF File**  
  A static version of the dashboard for users who do not have Power BI installed.

- 📊 **PBIT File (Power BI Template)**  
  For users with Power BI installed:
  - Explore the dashboard interactively  
  - Modify visuals and data  
  - Reuse the template with your own dataset  

## 🛠 Tools Used

- Power BI  
- DAX (Data Analysis Expressions)  
- Data Cleaning & Transformation  
- Data Visualization  

## 📌 Summary

This project provides insights into sales performance and customer behavior, enabling better data-driven decision making.
