# 📊 Super Sales Store Analytics - Power BI Project

This project showcases an **interactive Power BI dashboard** built on **Super Sales Store Analytics** data. It focuses on analyzing sales performance and customer behavior using key business metrics.

## 🚀 Key Features

- 📈 Developed an interactive dashboard using KPIs:
  - Total Sales  
  - Profit  
  - Quantity Sold  
  - Profit Margin  

- 🧹 Performed data cleaning and transformation to ensure accurate analysis
  
- Used DAX (Data Analysis Expressions) to create calculated measures and enhance data insights
  
- 📊 Created interactive visualizations:
  - Charts, Maps, Filters, and Slicers  
  - Regional sales trends  
  - Product performance  
  - Customer purchasing patterns  

- 🔮 Applied forecasting techniques to identify future sales trends and support data-driven decision making

## 🛠 Tools Used

- Power BI  
- Data Cleaning & Transformation  
- Data Visualization  
- Forecasting  

## 📌 Project Overview

The dashboard provides actionable insights to support **data-driven decision making** by identifying trends, patterns, and future opportunities in sales data.
